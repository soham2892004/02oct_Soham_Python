 Que.2-how will you remove last object from a list? Suppose list1 is  [2, 33, 222, 14, and 25], what is list1 [-1]?
 Ans.	user remove last object from a list using pop() method.pop list method is removes the element at the specified index and returns it. if no index is provided, it removes and returns the last element by default.
		when list1 is [2,33,222,14,25],so list1[-1] value is 25.
