Que-How will you randomizes the items of a list in a place?
Ans.
    step-1:
        importing random module.

    step-2:
        create such as a list.

    step-3:
        shuffle to list using shuffle function in random module.

    step-4:
        displayed randomly shuffle list.

    Example.,
        import random

        Sample_list=[101,125,135,155,175,185]

        random.shuffle(Sample_list)

        print(Sample_list)
